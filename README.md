# FordPass Rewards

### Requisitos

* Deve ter Docker e Docker Compose instalados em seu ambiente de desenvolvimento.

### Installation

1. Docker container 
   ```sh
   docker-compose up
   ```
2. Build Application
   ```sh
   mvn clean install
   ```

Stack
- Java 17; 
- Spring Boot;
- Maven;
- PostgreSQL;
- Docker;
- IntelliJ IDEA Community;
- Git.
